# Mixed Messages

Mixed Messages is a basic interactive website that generates a random message by pushing a button.

## Installation

Simply download, or copy and paste, the files into the same folder and open the HTML file in a browser and push the button.

```bash
1. Navigate to directory containing files
2. open ./index.html
```
